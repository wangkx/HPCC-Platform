How to spray file?

To spray a file:
  1. Use the drop-list to select the Machine/dropzone drop down menu where the source file is located. Verify IP Address below Machine/dropzone drop down menu.
  2. Verify the file path of source file to spray. Add subpath as needed.
  3. Select the file to be sprayed using the Choose File button.
  4. Fill in the Record Length.
  5. Select a destination Group using the Group drop down menu.
  6. Specifiy a label for sprayed file.
  7. Specifiy a prefix for sprayed file.
  8. Select the options:
    • Overwrite: Check this box to overwrite files of the same name.
    • Compress: Check this box to compress the files
  9. Press on Submit button Now to complete the file spray.
