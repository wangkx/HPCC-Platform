How to upload file?

For smaller data files, maximum of 2GB, you can use the upload/download file utility. To upload a large file for processing to your virtual machine, you will need a tool that supports the secure copy protocol.

To upload a small file:
  1. Select a dropzone from the Machine/dropzone list. Verify IP address and local file path.
  2. Press the Browse button to browse the files on your local machine.
  3. Using the Choose File button, open a file selection dialog.
  4. Select the file to upload and then click Open button. The file you selected should appear in the Select a file to upload field.
  5. Press on Upload Now to complete the file upload.

To download a file, click the file name inside the file list.
